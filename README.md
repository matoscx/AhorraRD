# AhorraRD
Sistema de optimización de compras con Python

Desarrollo de un sistema en Python para optimizar compras de productos básicos por barrio, identificando el colmado más conveniente y calculando el ahorro estimado frente al precio promedio del mercado.

El proyecto incluye:
Procesamiento y limpieza de datos
Cálculo de precios promedio por zona
Comparación de costos por producto
Generación de métricas de ahorro
Recomendación automática de sustitutos
Cálculo de costo total optimizado

Se utilizaron estructuras de datos y análisis exploratorio para apoyar la toma de decisiones basadas en datos.
Herramientas: Python, Pandas y Jupyter Notebook.
